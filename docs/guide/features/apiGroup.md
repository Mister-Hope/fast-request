# API 分组

版本要求: <Badge text="2022.2.1"/>

目的:一个 API 下保存不同组合的参数

[_B 站教程_](https://www.bilibili.com/video/BV1zU4y1S7pC?share_source=copy_web&vd_source=c46db3e7c134b1948dabbea9717a72ac)
